# Trademarks

This document outlines the trademark, brand usage rules and naming policies for the **PHPShift** open-source framework, created, developed and maintained by **Irakli Gzirishvili**.

## 1. Trademark

The name **PHPShift** is a trademark by the author, intended to identify the unique identity of this framework and distinguish it from other PHP-related framework, tools and libraries.

While not currently registered as a formal trademark, the name **PHPShift** is actively protected as a brand associated exclusively with the original framework repositories and its official distribution channels.

You may refer to PHPShift, but **you may not**:

- Use the name "PHPShift" in your own project names, packages, or marketing materials in a way that suggests affiliation.
- Create forks, clones, or modified distributions under the "PHPShift" name or any confusingly similar name.
- Use the PHPShift logo, identity, or branding for your project, or any other project, product, or organization without express permission from the author.

## 2. Disclaimer Regarding "PHP"

**PHPShift** is an independent framework using **PHP** language, but it is **not endorsed by or affiliated with the PHP Group** in any way.
The use of "PHP" in the name "PHPShift" is strictly nominative and intended solely to indicate that this framework is built for the PHP projects.

The source code for PHPShift is licensed under the [MIT License](https://github.com/phpshift/PHPShift/blob/main/LICENSE).

PHPShift™ framework does not ship, alter, or redistribute the PHP interpreter or its source code in any form. It is a framework built using the PHP and Python programming languages and as such, the name of this framework is valid for this use.

## 3. Brand Integrity & Community Respect

The author of PHPShift™ framework welcomes community interest, discussion and feedback. However, preserving the identity and purpose of the framework is essential.

**Prohibited actions include:**

- Rebranding the framework with a new name and distributing it as a separate or competing product.
- Registering domains, organizations, packages, or plugins that include "PHPShift" without explicit permission.
- Implying official association or endorsement in promotional materials.
